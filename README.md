1) Building binary debian package.
Execute the below command with root privileges.
Notice that we do not support binaries generated by this method.

  # make

ex) If you build a binary debian using a compress file  hfcldd-4.50.20.4000-1_3.19.0-25.ubuntu.amd64.src.tgr.bz2
    execute this command.

  # tar -jxf hfcldd_4.50.20.4000-1_3.19.0-25.ubuntu.amd64.src.tar.bz2
  # cd hfcldd_4.50.20.4000-1_3.19.0-25.ubuntu.amd64.src
  # make

2) You can install the debian package generated in the same directory on excuting 1).

ex)
  # ls *deb

    hfcldd_4.50.20.4000-1_3.19.0-25.ubuntu.amd64.deb
