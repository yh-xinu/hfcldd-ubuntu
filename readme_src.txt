1) バイナリ作成方法

root権限で以下を実行します。
尚、作成したバイナリに関しては、サポートの対象外となりますのでご注意下さい。

  # make

ex) ソースの圧縮ファイル hfcldd_4.50.20.4000-1_3.19.0-25.ubuntu.amd64.src.tar.bz2 を使用して
動作中のカーネルバージョンで、アーキテクチャ x86_64 に対するバイナリを作成する場合。

  # tar -jxf hfcldd_4.50.20.4000-1_3.19.0-25.ubuntu.amd64.src.tar.bz2
  # cd hfcldd_4.50.20.4000-1_3.19.0-25.ubuntu.amd64.src
  # make

2) 1)のコマンドを実行したディレクトリにdebianパッケージが格納されます。

ex)
  # ls *deb
    hfcldd_4.50.20.4000-1_3.19.0-25.ubuntu.amd64.deb
